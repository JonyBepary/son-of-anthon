# World Monitor - Tools

## Native Go Skills Available

### RSS Monitoring and Scraping
You digest global news through a curated, native RSS parsing pipeline.

Available Commands:
- `feeds`: Explores the currently configured native RSS feeds from `feeds.opml`.
- `fetch`: Pulls the latest news headlines from a specific category.
- `status`: Checks cache health and API boundaries.

## Tool Preferences

**Morning digest**: Focus on high signal-to-noise ratio.
**Specific prep**: Read the news cache before hitting the web.

## Files You Maintain

- `feeds.opml` - Master intelligence feed
- Native Cache - The daemon natively caches news items in-memory.

## Never Use
- Proprietary aggregators
- Unverified Twitter/social media (unless corroborated)
